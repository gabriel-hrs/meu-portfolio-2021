Afronte

Mobile app + Wearabledera website

This project was created during the 6th semester of the Digital Design graduation at Universidade Anhembi Morumbi, by the Afronte group in 2018. The project emerged as an alternative to the security of the black woman in Brazil. Where would be developed wearable accessories with technologies that provide the location of the woman, even a contact by direct message or call to the Women's Police Station.
During the project, research was carried out for the analysis and creation of the project proposal. As As a result, the visual part of the project was created, containing moodboard, iconography, color palette, typography, wireframe, wireflow and app layout. In addition to the conceptual part and research results like personas and user journey.
The accessory was created with Arduino, integrating with the application developed with the MIT App Inventor.

Mural

Mobile app e Website

This project was created during the 8th semester of the Digital Design graduation at Universidade Anhembi Morumbi, by the Mural group in 2019, as TCC of the course. The project emerged as a methodology that would aggregate the teaching of history in schools in Brazil.
During the project, research was carried out for the analysis and creation of the project proposal. As As a result, the visual part of the project was created, containing moodboard, iconography, color palette, typography, wireframe, wireflow and app layout. In addition to the conceptual part and research results like personas and user journey.
Pieces were created such as a scientific poster, an animation explaining the project and its products, a Canvas containing the project's business model, a web platform to provide the app, where it is currently only available for Android (under development). the platform web was created using the WordPress platform and the application was created using the Apache platform Cordova, HTML, CSS, JavaScript, plus Firebase database integration.
